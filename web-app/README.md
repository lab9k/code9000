Web application
====================
----------------

Getting Started
-------------
----------

####  Requirements

- yarn  (^v0.22.0)

####  Installation

To install the webapplication, first go the the folder "web-app". Then use yarn to install the required modules.

```
$  cd web-app/
$  yarn install
$  yarn start && yarn watch-css
```




Copy .env.example to .env, and change the URL to the one of your API.
```
$  cp .env.example .env
$  nano .env
$  yarn start
```
