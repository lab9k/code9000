Web application
====================
----------------

Getting Started
-------------
----------

####  Requirements

- yarn  (^v0.22.0)

####  Installation

To install the webapplication, first go the the folder "webapp". Then use yarn to install the required modules.

```
$  cd webapp/
$  yarn install
```


Copy .env.example to .env, and change the URL to the one of your API.
```
$  cp .env.example .env
$  nano .env
$  yarn start
```
